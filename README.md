# Project-1
This file includes files for Project #1 CIVE 202. For Project #1, The client, UNMC Water, Climate and Health Group, is collecting air quality data throughout the state of Nebraska. AirPurple monitors collected data from 2024 to 2025, which needs to be analyzed and interpreted. UNMC wants to know of any potential air quality hazards and their locations, as well as the effects of environmental factors on air quality.
